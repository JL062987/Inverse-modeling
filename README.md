This repository contains the code used in the following works:

[1] Justin Lien, Moment-Based Empirical Reconstruction of Stochastic Dynamical Systems with Temporally Correlated Noise and Its Applications to Prediction and Causal Inference, Ph.D. thesis, Tohoku University, 2026.
